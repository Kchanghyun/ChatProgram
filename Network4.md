# 3계층의 기능
다른 네트워크 대역 즉, 멀리 떨어진 곳에 존재하는 네트워크까지 어떻게 데이터를 전달할지 제어하는 일을 담당\
발신에서 착신까지의 패킷의 경로를 제어

---

classful IP주소 -> classless IP -> 사설IP와 공인IP

현재는 classless IP와 사설IP, 공인IP(외부와 통신할 때)를 같이 쓰고 있다.

>사설IP(컴퓨터, 스마트폰)가 공인IP(공유기)를 통해 인터넷에 데이터(웹툰)를 요청하면 인터넷은 공인IP(공유기)에게 데이터를 전달하고 사설IP(컴퓨터, 스마트폰)에게는 공인IP(공유기)가 전달해준다. - NAT(Network Address Translation)이라는 기술을 통해 IP 변환

실제 인터넷 세상에서는 공인IP로만 통신\
외부 네트워크 대역에서는 사설IP 대역이 보이지 않는다.

사설IP가 공인IP를 통해 나갈 때 NAT Table에 기록이 남지 않으면 공인IP에만 데이터가 들어오고 사설IP에게까지 데이터가 들어가지 않는다.

### 특수한 IP 주소
- 0.0.0.0(Wildcard) : 나머지 모든 IP
- 127.0.0.1 : 나 자신을 나타내는 주소
- 게이트웨이 주소 : 어딘가로 가려면 일단 여기로..(공유기IP)
